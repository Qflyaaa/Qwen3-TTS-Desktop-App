# Qwen3-TTS 桌面应用

基于 Tkinter 的 Qwen3-TTS 语音合成桌面应用。

## 功能特性

### 1. 语音克隆并保存音色特征
- 上传参考音频文件
- 输入参考文本
- 提取并保存音色特征
- 支持完整模式和X-Vector模式

### 2. 加载音色特征并朗读文本
- 选择已保存的音色
- 输入或加载文本文件
- 支持语气控制（可选）
- 生成语音并保存

### 3. 定制音色并朗读文本
- 使用音色设计模型
- 通过自然语言描述定制音色
- 生成指定风格的语音

### 4. 文件管理
- 音色库管理
- 生成历史查看
- 统计信息
- 文件删除和播放

## 安装

1. 确保已安装 Python 3.8+
2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 确保 Qwen3-TTS 模型已下载到工作空间根目录的 `Qwen3-TTS` 文件夹中

## 运行

```bash
python app.py
```

## 使用说明

### 语音克隆
1. 切换到"语音克隆"标签页
2. 点击"浏览"选择参考音频文件
3. 输入或加载参考文本
4. 输入音色名称
5. 选择模式（推荐完整模式）
6. 点击"开始提取特征"

### 文本朗读
1. 切换到"文本朗读"标签页
2. 选择已保存的音色
3. 输入或加载文本
4. （可选）输入语气控制指令
5. 选择语言和输出目录
6. 点击"生成语音"

### 音色设计
1. 切换到"音色设计"标签页
2. 输入或加载文本
3. 输入音色描述
4. 选择语言和输出目录
5. 点击"生成语音"

### 文件管理
1. 切换到"文件管理"标签页
2. 查看音色库、生成历史和统计信息
3. 可以删除音色或音频文件
4. 可以播放生成的音频

## 注意事项

- 首次使用需要加载模型，可能需要一些时间
- 建议使用 GPU 加速（需要安装 CUDA 版本的 PyTorch）
- 音频文件建议使用 WAV 格式
- 文本长度建议不超过 5000 字符

## 许可证

本项目基于 Qwen3-TTS 模型开发。
